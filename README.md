# Лабораторная работа 2. Управление инвентарем в ролевой игре

Я реализовала систему управления инвентарем для  игры.

Система позволяет:
- добавлять в инвентарь различные типы предметов: оружие, броня, зелья, квестовые предметы;
- учитывать уникальные свойства предметов (урон у оружия, защита у брони, лечение у зелий);
- использовать или экипировать предметы;
- получать информацию о текущем состоянии инвентаря;
- улучшать и комбинировать предметы.
